# Uso de Redes Sociales entre Estudiantes – Análisis Exploratorio (EDA)

## Descripción del proyecto
Este proyecto analiza el uso de redes sociales entre estudiantes y su posible impacto en distintos aspectos de su vida cotidiana y bienestar. El dataset incluye información sobre:  

- Horas promedio de uso diario en redes sociales (`Avg_Daily_Usage_Hours`)  
- Plataformas más utilizadas (`Most_Used_Platform`)  
- Nivel académico (`Academic_Level`)  
- Sueño (`Sleep_Hours_Per_Night`)  
- Salud mental (`Mental_Health_Score`)  
- Puntaje de adicción a redes sociales (`Addicted_Score`)  
- Conflictos relacionados con redes sociales (`Conflicts_Over_Social_Media`)  
- Otros datos demográficos: edad, género, país y estado de relación  

La temática del dataset permite explorar patrones y relaciones entre los hábitos digitales de los estudiantes y factores personales, sociales y académicos, ofreciendo una visión integral de cómo las redes sociales se integran en su rutina y bienestar.

---

## Objetivos del análisis
- Identificar las plataformas más populares según edad, género y país.  
- Analizar el promedio de horas diarias en redes sociales y su relación con plataformas específicas.  
- Explorar la posible relación entre el uso de redes sociales y la cantidad de sueño.  
- Examinar correlaciones entre uso de redes sociales, salud mental, adicción y conflictos interpersonales.  
- Visualizar tendencias, detectar valores atípicos y comprender patrones de comportamiento digital.

---

## Estructura del dataset
| Columna | Descripción |
|---------|------------|
| `Student_ID` | Identificador del estudiante |
| `Age` | Edad del estudiante |
| `Gender` | Género del estudiante |
| `Academic_Level` | Nivel académico (High School, Undergraduate, Graduate) |
| `Country` | País de residencia |
| `Avg_Daily_Usage_Hours` | Promedio de horas diarias en redes sociales |
| `Most_Used_Platform` | Plataforma más utilizada |
| `Affects_Academic_Performance` | Impacto percibido en el rendimiento académico (Sí/No) |
| `Sleep_Hours_Per_Night` | Horas de sueño por noche |
| `Mental_Health_Score` | Puntaje de salud mental (1-10) |
| `Relationship_Status` | Estado de relación sentimental |
| `Conflicts_Over_Social_Media` | Número de conflictos relacionados con redes sociales |
| `Addicted_Score` | Puntaje de adicción a redes sociales (1-10) |
